# Frontend - Hotel Room Manager

This directory will contain the Web Application for:
1. **Staff Dashboard**: Real-time monitor of room statuses (Occupied, Vacant, Power ON/OFF).
2. **Guest Self Check-in**: A mobile-friendly interface for guests to scan a QR code and check into their room.

## Tech Stack Recommendation
- React (via Vite) or Next.js
- TailwindCSS (Premium UI/UX required)
- WebSockets for real-time room status updates
