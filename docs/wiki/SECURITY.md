## 🛡️ นโยบายการรักษาความปลอดภัย (Security Policy)

ความปลอดภัยของข้อมูลผู้เข้าพักและระบบควบคุมฮาร์ดแวร์ของโรงแรมเป็นสิ่งที่เราให้ความสำคัญสูงสุด

### รุ่นที่รองรับการอัปเดตความปลอดภัย (Supported Versions)

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

### การรายงานช่องโหว่ (Reporting a Vulnerability)

หากคุณพบช่องโหว่ด้านความปลอดภัยในระบบ โปรดอย่าเปิดเผยต่อสาธารณะ ให้ส่งอีเมลรายงานมาที่ทีมผู้ดูแลระบบ (Admin) หรือส่งแจ้งผ่านช่องทางสื่อสารหลักของโปรเจกต์ เราจะดำเนินการตรวจสอบและแก้ไขภายใน 48 ชั่วโมง

## 🔒 การปฏิบัติตาม พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล (PDPA Compliance)

ระบบ Hotel ECS ได้รับการออกแบบโดยคำนึงถึง Privacy by Design และสอดคล้องกับมาตรฐาน PDPA ของประเทศไทย

### 1. การเก็บรวบรวมข้อมูลอย่างจำกัด (Data Minimization)
ระบบจะเก็บรวบรวมเฉพาะข้อมูลที่จำเป็นต่อการทำ Self Check-in และการเข้าพักเท่านั้น เช่น:
- หมายเลขการจอง (Booking ID)
- ชื่อ-นามสกุล (สำหรับแสดงผลที่ตู้สาขา PBX และต้อนรับ)
- เบอร์โทรศัพท์ / อีเมล (สำหรับส่ง OTP หรือการติดต่อกรณีฉุกเฉิน)
- ข้อมูลการ Check-in/Check-out (Timestamp)

### 2. การเข้ารหัสข้อมูล (Data Encryption)
- **Data in Transit**: ข้อมูลทั้งหมดที่ส่งระหว่าง Frontend, Backend และ PBX Connector จะถูกเข้ารหัสผ่านโปรโตคอลความปลอดภัยมาตรฐาน (HTTPS/WSS)
- **Data at Rest**: ข้อมูลส่วนบุคคลในฐานข้อมูลจะถูกเข้ารหัส เพื่อป้องกันการเข้าถึงจากผู้ไม่หวังดี

### 3. ระยะเวลาการจัดเก็บข้อมูล (Data Retention)
ข้อมูลผู้เข้าพักจะถูกจัดเก็บไว้ในระบบเพื่อใช้ในการ Audit และอ้างอิงตามกฎหมายที่พักแรม เป็นระยะเวลาตามที่กฎหมายกำหนด หลังจากนั้นข้อมูลส่วนบุคคล (PII) จะถูก Anonymized หรือลบออกจากระบบโดยอัตโนมัติ

### 4. สิทธิของเจ้าของข้อมูล (Data Subject Rights)
ผู้เข้าพักสามารถร้องขอให้โรงแรม:
- เข้าถึง หรือขอสำเนาข้อมูลส่วนบุคคล
- ลบ หรือทำลายข้อมูลส่วนบุคคล (Right to be forgotten) เมื่อสิ้นสุดระยะเวลาการเข้าพักและไม่มีข้อผูกพันทางกฎหมาย
- ถอนความยินยอมในการประมวลผลข้อมูล

### 5. การจัดการการยินยอม (Consent Management)
ในขั้นตอนการทำ Self Check-in ระบบจะมีหน้าต่างข้อตกลงและเงื่อนไข (Terms of Service) และนโยบายความเป็นส่วนตัว (Privacy Policy) ให้ผู้เข้าพักอ่านและกดยืนยัน (Consent) อย่างชัดเจนก่อนดำเนินการต่อเสมอ
