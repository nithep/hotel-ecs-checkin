## 📅 ไอเดียการแบ่งตอน (Episode Breakdown)

### 🔴 EP.1: "ลุกขึ้นสู้กับตู้สาขาอายุ 20 ปี" (The Legacy Code Challenge)
*   **แกนเรื่อง (Story arc):** 
    - เปิดฉากด้วยความล้าสมัยของโปรแกรม "Room Manager" บน PC ที่เก่าและเสี่ยงต่อการพังตลอดเวลา
    - แนะนำตัวเอกใหม่ **Raspberry Pi 4** ที่จะมารับบทเป็นผู้ควบคุมตู้สาขา Phonik PBX
*   **จุดดราม่าหน้างาน:** การแกะโปรโตคอล ASCII ที่ไม่มีคู่มือชัดเจน การยิงคำสั่งเปิดไฟห้องพักสำเร็จครั้งแรก
*   **คีย์เวิร์ดเรียนรู้:** Serial/TCP, Relay Control, PBX Commands.

### 🔴 EP.2: "อินเทอร์เน็ตพัง! บั๊ก IP ชนและวิธีกู้ชีพหัวเกียร์" (The Field Network Battle)
*   **แกนเรื่อง (Story arc):**
    - การลงพื้นที่จริงเพื่อย้ายบอร์ด Pi ไปต่อเข้าระบบเน็ตเวิร์กหน้างาน
    - เกิดปัญหา Pi หลุดการเชื่อมต่อทาง SSH และเบราว์เซอร์เปิดเว็บไม่ได้เพราะ **Wi-Fi และ LAN ชนกันแย่งทางวิ่งข้อมูล (Routing confusion)**
    - การแก้ปัญหาแบบสั่งตัดด้วยการปิด Wi-Fi ในระดับบอร์ดคอมพิวเตอร์และบังคับวิ่ง LAN ท่อเดียว
*   **คีย์เวิร์ดเรียนรู้:** IP Conflict, nmcli, disable-wifi, mDNS (`pi-4.local`).

### 🔴 EP.3: "เว็บล่มหน้างาน! วิธีสร้างอุโมงค์ทะลุกำแพงด้วย Cloudflare Tunnel" (How we solved 502 Bad Gateway)
*   **แกนเรื่อง (Story arc):**
    - เมื่อผู้ให้บริการอินเทอร์เน็ตเปลี่ยน IP (DHCP) หน้างาน ทำให้พอร์ตนอกหลุดจนเว็บเข้าไม่ได้ (502 Bad Gateway)
    - วิธีแก้ปัญหาที่ชาญฉลาดโดยไม่ต้องเปิดพอร์ตเราเตอร์หรือเช่า Fixed IP ด้วยการติดตั้ง **Cloudflare Tunnel** 
    - การผูก Docker Container Name แทน IP Address ป้องกันปัญหา IP เปลี่ยนถาวร
*   **คีย์เวิร์ดเรียนรู้:** Zero Trust, Cloudflare Tunnel, Docker Compose, CNAME.

### 🔴 EP.4: "สมองกล Copilot: สร้าง AI ซ่อมตัวเองเมื่อคีย์โดนแบน" (Designing the Self-Healing AI Assistant)
*   **แกนเรื่อง (Story arc):**
    - ไอเดียการมี AI ถามตอบวิเคราะห์ปัญหาบนบอร์ดควบคุมเหมือน Cloudflare Ask AI
    - อุปสรรคครั้งใหญ่: คีย์ Gemini API ติดนโยบายความปลอดภัยและโควตาเป็น 0 (`limit: 0`) จนคุยกับ AI ไม่ได้
    - เบื้องหลังการเขียนสคริปต์ **Hybrid Gateway** เพื่อดึง OpenRouter (Gemini 2.5 Flash) มาใช้งานแทนด้วยราคาค่าประมวลผล 0.007 บาทต่อครั้ง พร้อมระบบออฟไลน์กู้ภัยวินิจฉัยโรค (Offline Fallback) ที่ไม่มีทางตาย
*   **คีย์เวิร์ดเรียนรู้:** Hybrid AI Gateway, API Quotas, Offline Diagnostics, OpenRouter integration.

## 🛠️ โครงสร้างสคริปต์ในแต่ละคลิป (Video Structure)
1.  **Hook (30 วินาทีแรก):** โชว์ปัญหารุนแรงหน้างานทันที (เช่น หน้าจอแดงแจ๋, สายไฟระโยงระยาง, โดเมนเข้าไม่ได้) แล้วตั้งคำถามกระตุ้นความสนใจ
2.  **The Journey (การเดินทาง):** พาทัวร์จุดเกิดเหตุและแกะโค้ดแก้ปัญหาร่วมกันทีละบรรทัด (Live debugging)
3.  **The Fix (การแก้ปัญหา):** อธิบายหลักการทางวิศวกรรมที่นำมาใช้แก้ปัญหาแบบเข้าใจง่าย (ใช้กราฟิก 2D อธิบาย)
4.  **UAT & Verification (ผลทดสอบ):** โชว์ผลลัพธ์ว่าไฟติดจริง ปลั๊กใช้ได้จริง และ AI ตอบคำถามได้ปกติ
5.  **Takeaway (ข้อคิดช่าง):** สรุปบทเรียนสำหรับช่างเทคนิคที่สามารถนำไปปรับใช้กับตู้คอนโทรลโรงงานหรือคอนโดของตนเองได้
