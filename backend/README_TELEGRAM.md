# คู่มือการตั้งค่า Telegram Bot สำหรับระบบอนุมัติความปลอดภัย (Hotel ECS)

ระบบ Hotel-ECS มีระบบส่งการแจ้งเตือนและการอนุมัติคำสั่งเสี่ยงสูง (High-Risk Commands) ผ่านทาง Telegram Bot เพื่อให้แอดมินหรือผู้ดูแลระบบสามารถควบคุมความปลอดภัยของตู้สาขาและระบบไฟผ่านมือถือได้ทันที

---

## 🛠️ ขั้นตอนการสร้าง Telegram Bot และตั้งค่า

### ขั้นตอนที่ 1: สร้าง Bot ด้วย BotFather
1. เปิดแอปพลิเคชัน Telegram แล้วค้นหาห้องแชทของ **@BotFather**
2. กดปุ่ม `Start` หรือพิมพ์คำสั่ง `/newbot`
3. ตั้งชื่อบอทของคุณ (เช่น `My Hotel ECS Bot`)
4. ตั้ง Username ของบอท ซึ่งต้องลงท้ายด้วยคำว่า `bot` (เช่น `my_hotel_ecs_bot`)
5. เมื่อสร้างเสร็จ BotFather จะส่ง **HTTP API Token** มาให้ (ตัวอย่าง: `1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ`) 
   - ให้คัดลอกโทเค็นนี้เก็บไว้สำหรับตั้งค่า `TELEGRAM_BOT_TOKEN`

### ขั้นตอนที่ 2: หา Chat ID
เนื่องจากบอทจำเป็นต้องรู้ว่าจะส่งแจ้งเตือนไปที่ห้องแชทใด (แชทส่วนตัว หรือกลุ่มแอดมิน):

#### กรณีแชทส่วนตัว (Direct Message)
1. ค้นหาบอทของคุณใน Telegram ตาม Username ที่ตั้งไว้ในขั้นตอนแรก
2. กดปุ่ม `Start` หรือพิมพ์ `/start`
3. บอทจะตอบกลับอัตโนมัติพร้อมแจ้งเตือน **Chat ID** ของคุณ (ตัวอย่าง: `987654321`)
   - ให้คัดลอกรหัสนี้เก็บไว้สำหรับตั้งค่า `TELEGRAM_CHAT_ID`

#### กรณีกลุ่มแอดมิน (Group Chat)
1. สร้างกลุ่มแอดมินใน Telegram และกดเชิญบอทของคุณเข้าร่วมกลุ่ม
2. ส่งข้อความเข้าไปในกลุ่ม เช่น `/info@your_bot_username`
3. บอทจะตอบกลับในกลุ่มพร้อมแจ้ง **Chat ID** ของกลุ่ม (ซึ่งจะมีเครื่องหมายลบนำหน้าเสมอ เช่น `-456789123`)
   - ให้คัดลอกรหัสแชทกลุ่มนี้เก็บไว้สำหรับตั้งค่า `TELEGRAM_CHAT_ID`

---

## ⚙️ การตั้งค่าในระบบ (.env)

เปิดไฟล์ `backend/.env` และเพิ่มข้อมูลที่คุณได้รับ:

```env
# โทเค็นบอทที่ได้จาก BotFather
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ

# ไอดีห้องแชท (แชทส่วนตัว หรือแชทกลุ่มที่มีบอทอยู่)
TELEGRAM_CHAT_ID=-456789123
```

จากนั้น ทำการรีสตาร์ทเซิร์ฟเวอร์ Backend ระบบจะเริ่มทำ Long Polling เพื่อรอคำสั่งและแจ้งเตือนทันทีครับ
